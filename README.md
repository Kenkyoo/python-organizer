# Organizador de Archivos

## Descripción

El proyecto Organizador de Archivos es una herramienta diseñada para simplificar el proceso de organización y gestión de archivos en sistemas operativos. Con esta herramienta, los usuarios pueden clasificar y ordenar sus archivos de manera eficiente, lo que facilita la búsqueda y acceso a los documentos necesarios.

## Funcionalidades

- Clasificación automática de archivos según sus extensiones.
- Creación de carpetas organizadas por tipo de archivo.
- Copia y movimiento de archivos a las carpetas correspondientes.
- Registro de cambios en un archivo de registro.
- Eliminación de archivos duplicados.
- Generación automática de archivos README.md para proyectos.

## Uso

Para utilizar el Organizador de Archivos, siga estos pasos:

1. Descargue y descomprima el archivo ZIP del proyecto.
2. Abra una terminal en el directorio del proyecto.
3. Ejecute el comando `python app.py` para iniciar la aplicación.
4. Siga las instrucciones en pantalla para organizar sus archivos.
